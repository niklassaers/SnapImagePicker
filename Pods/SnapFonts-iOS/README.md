#SnapFonts-iOS

A Swift port of https://github.com/CocoaPods-Fonts/OpenSans, encapsulating the fonts used in Snapsale's iOS app.


###Usage

```swift
import SnapFonts_iOS

let font = SnapFonts.gothamRoundedBoldOfSize(20)
```
